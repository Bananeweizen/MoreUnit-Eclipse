Update site

Please change your update-site to the new URL. Please uninstall older versions of MoreUnit first. The feature id has changed and you will get error messages if older versions are still installed when switching to version 2.0.1.

Use the following update site:
http://moreunit.sourceforge.net/update-site/

The old update-site http://moreunit.sourceforge.net/org.moreunit.updatesite/ will not contain any new releases and gets freezed with version 1.3.3.
