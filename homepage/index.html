MoreUnit makes it easy to write more unit tests!
Overview

MoreUnit is an Eclipse plugin that should assist you in writing more unit tests. This is a short overview of the features it provides, for further information take a look at the documentation section:

  • MoreUnit decorates classes which have a test case.
  • It marks methods in the editor which are under test.
  • You can jump to a test case/test method in the editor via the menu or a shortcut (Ctrl-J by default).
  • You can run a test case/test method from the class under test via the menu or a shortcut (Ctrl-R by default).
  • Rename classes/methods and MoreUnit will rename the corresponding test code too.
  • Move classes and MoreUnit will move the corresponding test cases.
  • You can generate a test method stub for the method under the cursor in the editor via the menu or a shortcut.
  • With the Mock plugin: select the dependencies of a class that you want to see mocked into a corresponding test case.

MoreUnit 3 is out! (Well, sort of...)

We just released MoreUnit 3.0.0.01. Yep, that's a lot of numbers!

The 01 part informs you that this version is a milestone on the road to the actual version 3.0.0 that we wanted to release. Unfortunately, we're quite busy right now and we do not progress as fast as we would like. Therefore we decided to deliver this major version incrementally.
That said, no worry: this version isn't less stable than any other one!

Now, what makes this version a major one? Mainly: we added basic support for all programming languages, as well as a more powerful way to define your test naming patterns. Also, we moved a lots of things internally. For a more complete list of changes, see the release notes.

And here is what will come before MoreUnit 3 is deemed "complete":

  • Addition of markers for all test files.
  • New templates to generate boiler-plate code for JMockit and Powermock, as requested by some users. (Don't hesitate to ask us for integrating other ones!)
  • Migration to Java 6.
  • Slimming cure: MoreUnit's download size will lose almost 2 MB.
