Missing testmethod view

MoreUnit offers a view to show the methods of the java file you are working on which do not have a testmethod yet. To use this view just open it: WINDOW → SHOW VIEW → OTHER... → MISSING TESTMETHODS.


This view offers 3 features:
  • Filter private methods
  • Filter getter/setter
  • Create testmethods for selected methods
